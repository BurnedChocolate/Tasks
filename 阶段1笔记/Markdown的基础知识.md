# Markdown的基础操作
## 📖笔记内容
### 1.初识Markdown
就定义来说，Markdown是一种轻量级标记语言。它以纯文本形式编写文档，具有易读、易写、易更改的特质。可以导出PPT、LaTex、HTML、Word、PDF、Epub、JPG等多种格式的文档。

对于个人使用体验而言，不得不说，在接触了Markdown并且自己用它尝编写了一些文字之后，我已经感觉这辈子都不需要再用敲字敲一天排版排半年的word了……真是非常好用，爱不释手。

首先，Markdown的语法很容易上手，读起来规整美观，没有过多庞杂的格式。其次，Markdown兼容HTML，可转换为HTML格式发布，并且可跨平台使用，许多网站都支持Markdown的语言格式。我以前搭建的个人写作博客，使用的语言就是Markdown。现在正式开始学习Markdown，有一种十分亲切的熟悉感……

### 2.Markdown基本语法
Markdown语法主要分为如下几大部分：
>标题，段落，区块引用，代码区块，强调，列表，分割线，链接，图片，反斜杠`\`，符号"`"。

下面进行漫长的编写格式和示例展示……（注意要写空格！）
#### 2.1标题
    # 一号标题
    ## 二号标题
    ### 三号标题
    #### 四号标题
    ##### 五号标题
    ###### 六号标题
    ####### 七号……欸没了

## 二号标题

所以说是没有七号标题的，打七个`#`标题字号也不会变得更小

另外还有一种写法是：

#### 2.2段落
其实Markdown的换行语法对我来说一开始用得是比较别扭的……一直要记得换行或者加空格，一开始学习的时候很容易就会忘记，导致preview之后要回来修改半天。不过用着用着也就习惯了……  
##### 2.2.1换行  
**Markdown的换行方法有两种：**
1. 在段落末尾添加两个空格符号
2. 在段与段之间添加一个空行

不过这个规则再前后有标题或者序号/不定序号之间似乎不成立，序号或标题（这里指用`#`打出的标题）会自动帮你换行。

由于空格比较难以演示，这里就不演示了……记住两种换行方法就好。
##### 2.2.2字体  
我们直接上字体演示：  

    *我是斜体*
    _我是斜体_
    **我是粗体**
    __我是粗体__
    ***我是粗斜体***
    ___我是粗斜体___

##### 2.2.3分隔线
用三个及以上的`*`或`-`都可以达成分隔线的效果，之间可以有空格，行内不能有其他字符。  

    ***
    * * *
    ---
    - - -

***
* * *
---
- - -

另外这里自己打出的分隔线和标题附带的分隔线有粗细差别，仔细看的话是可以分别出来的。顺便，要小心不要把连起来的减号打成标题了。

#### 2.4代码区块
代码区块其实在之前的部分已经使用多次了。  

##### 小区快，起到标记作用  

\`printf()\`  

`printf()`

##### 一大片代码，有两种方法可以使用  
1.在行前插入四个空格或一个Tab键  
2.在段落前后插入\`\`\`，这种方法可以用于标记该段代码使用的编程语言（不标也行）。  

第一种：

      int main()
      {
          printf("Hello, Geek!\n");
      }
      
int main()  
{  
    printf("Hello, Geek!\n");  
}  


第二种：

```c
int main()
{
   printf("Hello, Geek!\n");
}
```

\```c  
int main()  
{  
   printf("Hello, Geek!\n");  
}  
\```  

这里要注意不要和标号以及普通段落混在一起……否则会不能显示成代码块区。
